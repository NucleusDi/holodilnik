# Холодильник

Одностраничный сайт для общего холодильника с магнитами. Посетители перетаскивают картинку на холодильник, она сохраняется в `mems/` и приклеивается на выбранном месте. Лайки считаются по cookie. Админка доступна на `/adminka`.

## Локальный запуск

```bash
npm install
cp .env.example .env
npm start
```

Сайт будет доступен на `http://localhost:3000`.

## Настройки

Переменные окружения:

- `PORT` - порт приложения, по умолчанию `3000`
- `SESSION_SECRET` - длинная случайная строка для сессий
- `ADMIN_PASSWORD` - пароль админа, по умолчанию `admin-pass`
- `SITE_TITLE` - начальная надпись на холодильнике

Админка:

- адрес: `/adminka`
- пароль по умолчанию: `admin-pass`

В админке можно поменять текст или картинку заголовка, включить модерацию, одобрять, скрывать и удалять магниты.

## Деплой на Ubuntu 24.04

```bash
sudo apt update
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
npm install --omit=dev
cp .env.example .env
nano .env
npm start
```

Для постоянного запуска удобно использовать `systemd` или `pm2`, а домен `holodilnik.dimon.ru` проксировать через Nginx на локальный порт приложения.

Папки `mems/` и `data/` создаются автоматически рядом с сайтом. В `mems/` лежат картинки магнитов, в `data/holodilnik.sqlite` - позиции, лайки, статусы и настройки.
